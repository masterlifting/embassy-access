Запись возможна на дни, выделенные белым цветом. Серым цветом будет выделен выбранный день.

Зарезервируйте время для посещения по выбранному вопросу в правой колонке. Нажмите кнопку «Записаться на прием».

Уважаемые посетители! При работе с сайтом записи на прием необходимо использовать только навигацию сайта. Во избежание потери данных о записи не используйте кнопку браузера «На предыдущую страницу».

Уважаемый/ая, в настоящий момент на интересующее Вас консульское действие в электронной системе предварительной записи нет свободных мест на ближайшее время.

Обратите внимание, что для зарегистрированных в системе заявок запись на прием осуществляется полностью в АВТОМАТИЧЕСКОМ режиме, без возможности самостоятельного выбора даты и времени.

Сообщение о назначенном времени приёма и подтверждении прихода будет направлено Вам на адрес электронной почты, указанный при регистрации заявки. Подтверждение Вашего прихода является обязательным условием приёма.

ВЕСЬМА ВАЖНО! Если после первичной регистрации в системе электронной записи на приём в течение 30 дней Вы ни разу не проверили статус своей заявки, на ваш адрес электронной почты будут направлены пять предупреждений об автоматическом удалении заявки.

Будьте в курсе всех оповещений системы электронной очереди, ежедневно проверяйте Вашу электронную почту.
