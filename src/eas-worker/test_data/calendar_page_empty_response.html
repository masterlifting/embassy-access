СПИСОК ОЖИДАНИЯ

В настоящий момент на интересующее Вас консульское действие в системе предварительной записи нет свободного времени.

Для проверки наличия свободных мест и последующей записи на прием Вы можете проверять статус Вашей заявки не чаще одного раза в день.

Появление свободных мест для записи по интересующему Вас консульскому действию Вы можете увидеть:

- перейдя по ссылке, отправленной на электронную почту, указанную Вами при регистрации;

- выбрав на главной странице электронной системы записи на прием пункт «Проверка статуса записи на прием»;

- перейдя по ссылке Проверка статуса записи на прием.
